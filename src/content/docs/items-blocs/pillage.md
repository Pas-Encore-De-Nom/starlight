Le plugin **Pillage** introduit des mécaniques d'attaque et de défense pour rendre les raids plus tactiques et immersifs.

## Armes d'attaque

### 🧨 Dynamite
Explosif posable qui détruit les blocs dans une zone. Particulièrement efficace contre les [Avant-Postes](/starlight/claims/avant-poste/).

### 💣 TNT
La TNT classique, utilisable dans les zones de claim avec le plugin pillage actif.

### 💥 Explosif (Canon)
Canon à projectiles explosifs pour les attaques à distance.

### 👺 Gobelin
Mob du jeu de base utilisé comme unité d'attaque.

### 🤖 Golem Foreuse
Un Golem qui creuse les blocs devant lui en **3×3**. Parfait pour percer les défenses rapidement.

### 🔧 Drill Portative
Outil de minage rapide qui consomme des consommables. Permet de percer des tunnels.

### 🔓 Crochetage
Permet d'ouvrir les coffres protégés dans les [Avant-Postes](/starlight/claims/avant-poste/) grâce à un **mini-jeu** de crochetage.

<Aside type="caution">
Le crochetage ne fonctionne **pas** dans les [Bastions](/starlight/claims/bastion/) — les coffres y sont entièrement protégés.
</Aside>

## Défenses

Les mécaniques de défense sont en cours de développement. Elles seront définies pendant la phase de développement du plugin.
