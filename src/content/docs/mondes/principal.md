Le **Monde Principal** est le cœur du serveur. C'est ici que se déroulent les guerres de factions, les claims et le PvP.

## Zones

| Zone | Description |
|---|---|
| **Spawn** | Point d'arrivée des joueurs. Zone protégée, aucun PvP. |
| **Warzone** | Zone de PvP libre entourant le spawn. Aucun claim possible. |
| **AP (Avant-Poste)** | Zones claimables par les factions |

## Caractéristiques

- **Pas de mobs** — Aucun mob naturel dans ce monde
- **Chunks pleins** — Les chunks autour du Spawn, de la Warzone et des Avant-Postes sont pré-générés et remplis

<Aside type="note">
L'absence de mobs dans le monde principal incite les joueurs à utiliser le [Monde Farm](/starlight/mondes/farm/) et le [Monde Minage](/starlight/mondes/minage/) pour progresser.
</Aside>

## Navigation

Depuis le Spawn, les joueurs peuvent accéder à tous les autres mondes via les portails ou commandes disponibles.
