# Changelog

All notable changes to GOAT will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0] - 2026-03-01

### Added

- Système de factions avec rôles (Chef, Officier, Membre, Recrue)
- Système de claims (Nature, Découvert, Avant-Poste, Bastion)
- Arbre de compétences avec 3 branches (Combat, Récolte, Support)
- Runes et Artefacts légendaires
- Banque de faction avec coffres partagés
- Système de puissance dynamique
- Relations diplomatiques entre factions (Allié, Neutre, Ennemi)
- Plugins : XP, Market, Spawner, Jobs, Événements, Quêtes
- Systèmes annexes : Casino, Mise à Prix PvP, Monitoring
- 4 mondes distincts : Principal, Minage, Farm, Lobby
- Système de surveillance et de logs d'administration

### Changed

- Structure initiale du serveur établie

### Fixed

- Aucun correctif pour la version 1.0
